Objetivo da fase

Criar o design system base, componentes reutilizáveis e identidade visual inicial do Lar dos Anjos Pet para portal público, área do doador e painel administrativo.

Correções obrigatórias
Não acoplar componentes UI a regras de negócio.
Componentes devem ser reutilizáveis em apps/web e apps/admin.
Não criar páginas finais nesta fase.
O design system deve considerar futura adaptação para app mobile.
Não implementar fluxo financeiro, Pix, Asaas, dashboard, portal de transparência ou área real do doador.
Não criar dependências desnecessárias com banco de dados nesta fase.
O que não implementar agora

Não implementar fluxos reais de doação, APIs, dashboards, dados dinâmicos, pagamentos, Pix, Asaas, portal de transparência, animais, campanhas, login avançado ou área do doador completa.

Todos os inputs devem ter label acessível.
Botões devem ter foco visível.

Menus devem funcionar com teclado.
Componentes interativos devem ter estados:
normal;
hover;
focus;
disabled;
loading;
erro, quando aplicável.
Não depender apenas de cor para indicar estado.
Componentes devem funcionar em mobile.
